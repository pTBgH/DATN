## 🎯 KEY FEATURES

Badge Component:
  • 6 variants: default, success, warning, error, info, primary
  • 3 sizes: sm, md, lg
  • Flexible and extensible

Button Component:
  • 5 variants: primary, secondary, outline, ghost, danger
  • 3 sizes: sm, md, lg
  • Loading state with spinner
  • Disabled state handling

Card Component:
  • Main container with hover effects
  • CardHeader (title, description, actions)
  • CardContent (body)
  • CardFooter (actions)

Expandable Component:
  • Native HTML <details> element
  • No JavaScript overhead
  • Smooth open/close transitions
  • Customizable summary text

## 🚀 IMPLEMENTATION EXAMPLES

Badge Display:
  <Badge variant="primary">Status</Badge>
  <Badge className={getStatusBadgeClass(status)}>{status}</Badge>

Card Usage:
  <Card hover>
    <CardHeader title="Title" description="Subtitle" />
    <CardContent>Body content</CardContent>
  </Card>

Formatter Usage:
  {formatDate(date, 'relative')}     // "5d ago"
  {truncateText(text, 150)}          // "Text..."
  {formatSalary(10M, 20M)}           // "$10M - $20M"

Expandable Content:
  <Expandable summary="Show more">
    Long content...
  </Expandable>
