# ZTA Cosign Keys

Generated by `scripts/zta-cosign-keygen.sh`.

| File | Purpose | Committed? |
|---|---|---|
| `zta.key` | Private signing key (ECDSA P-256) | ❌ **NEVER** — gitignored |
| `zta.pub` | Public verification key (PEM) | ✅ Yes — verifiers pull from here |

The public key is also published to cluster ConfigMap:
- `kubectl get cm -n security zta-cosign-public-key -o yaml`

For details see `knowledge-base/26-image-provenance.md`.
